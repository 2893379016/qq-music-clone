# QQ音乐风格在线音乐网站 🎵

一个功能完整的在线音乐流媒体网站，模仿QQ音乐的核心功能。

## 功能特性 ✨

- 🎵 音乐播放器（播放/暂停、上一曲/下一曲、音量控制、进度条拖拽）
- 📱 响应式设计，支持移动端和桌面端
- 📋 歌曲列表展示和搜索功能
- 🎤 正在播放歌曲信息显示
- 📊 播放历史记录
- ⬆️ 歌曲上传功能（支持MP3格式）
- 💾 本地数据库存储歌曲信息

## 技术栈 🛠️

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Node.js, Express.js
- **数据库**: SQLite
- **文件存储**: 本地文件系统
- **部署**: GitHub Pages

## 快速开始 🚀

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```

### 访问网站
打开浏览器访问: http://localhost:3000
